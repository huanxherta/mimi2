# Xiaomi AI Studio Claw Automation Control\n\nA web control panel for Xiaomi AI Studio Claw automation: environment reset, credential and key management, and an OpenAI-compatible relay.\n\n## Features\n\n### Web control panel\n- Modern web UI\n- Live status\n- Task logs\n\n### User management\n- Bulk import of Xiaomi credentials\n- Multiple credential formats (auto-detected)\n- User switching\n- **Multi-account OC pool** with automatic round-robin load balancing\n\n### Automation\n- One-click environment reset\n- SOUL template restore\n- Environment variable backup\n- **Automatic 401 retry** — on key expiry, transparently switch to next valid key and refresh the expired one in background\n\n### Key management\n- Automatic MIMO API key extraction\n- Key validation via chat API probing\n- **Background key monitor** — every ~50 minutes validates all OC keys; invalid ones are auto-refreshed via Claw\n- **Multi-key rotation** — supports multiple Xiaomi accounts, each with its own OC key, rotated per-request\n\n### OpenAI-compatible relay\n- OpenAI API–compatible surface (`/v1/chat/completions`, `/v1/models`)\n- Automatic key injection for upstream MIMO\n- Model name mapping\n- Streaming support\n\n## Quick start\n\n### 1. Run the web panel\n\nFrom the project directory:\n\n```bash\npython claw_web.py\n```\n\nIf `python` is not available, try `python3` or on Windows `py -3`.\n\nOpen: http://localhost:10060\n\nThe same port **10060** exposes an OpenAI-compatible API: set the client `base_url` to `http://localhost:10060/v1`.\n\n### 2. Import Xiaomi credentials\n\nIn the UI: **User management** → **Bulk import**.
